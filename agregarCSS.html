Como agregar CSS al HTML

Existen 3 formas de agregar CSS al HTML

  1. CSS Externo: Crear una hoja de css por fuera y conectarla en el head link rel="stylesheet" href="mystyle.css"
  2. CSS Interno: Crear con una etiqueta style de css y ir agregando estilos ahi dentro esto se crea dentro de la cabezera
  3. CSS en Linea: Esto es poniendo CSS como atributo a las etiquetas de HTML, esto debe ser moderado ya que se vera mucho desorden

Ten en cuenta que si se aplican estilos Internos pero se tiene ligado un CSS Externo se aplicara el del interno ya que este esta invodado mas abajo que el externo, de esta manera se trabaja con CSS en cascada de arriba a abajo se van aplicando los estilos.
